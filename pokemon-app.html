Pokemon Team Builder

School Project: Web Development

Context

So the secret's out: I'm a huge Pokemon fan. In the Pokemon world, trainers test their teams against other trainers in the hopes of someday becoming the Pokemon champion. This app helps the uncertain trainer begin their journey.

Using the React Framework

This app is based in React, a JavaScript library for building user interfaces. This app keeps track of a list of Pokemon and allows the trainer to (1) filter the Pokemon via their size or type (2) sort the Pokemon by weight and (3) add Pokemon to and from an aggregator. Once they have their team of 6, trainers can start their journey!

Takeaways

From this project, I learned how to use JavaScript's functional programming to communicate between Components. I also learned how to organize Component hierarchies and "lift up" State, as React likes to say. See the code or, better yet, play around with the app yourself!
